## Milestone 1 Goal

Deliver a no-terminal MVP where a non-technical user can complete design-to-export for enclosure/mount workflows.

## M1 Exit Gates

- >= 80% pilot users complete design-to-export without terminal help
- median time-to-first-valid-design < 10 minutes
- export completion rate >= 85%

## Current Sprint Focus

- [ ] App/backend IPC contract complete and stable
- [ ] Guided form flow fully replaces raw YAML editing in core path
- [ ] Versioned project save/load (`.rfa.json`) used end-to-end
- [ ] Deterministic exports (`STEP`, `STL`, report) available in one-click flow
- [ ] Rule-check output shown as actionable errors/warnings
